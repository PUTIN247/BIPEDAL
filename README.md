How to Use This Test
Upload the code.

Open Serial Monitor (Set baud to 115200).

Type w and hit Enter. The Hip should swing forward.

Type e and hit Enter. The Knee should bend (increasing the < angle).

Type d repeatedly. The Knee should extend but stop exactly at 0 (vertical leg) and refuse to go negative (hyperextension).
